About My URL shortner

I created this site to practice making a website that had a super simple concept but was more complete than my previous projects. This meant having unit tests, a CI pipeline that would ensure the tests were passing before allowing PRs to be merged, as well as deploying the app to a non-Heroku platform.

Tech stack used:

  • cookiecutter-flask - provided a great flask template with all the bells and whistles packaged in.
  • flask - the framework of choice for the backend. I started learning flask after I got laid off due to Covid-19 and have found it pretty easy to learn. It doesn't come with a lot of built in functionality, but it has great support for extensions. I found
  • MySQL - the database of choice for this project. I've used MySQL in university, as well as at my job at Ritual so I'm quite comfortable with writing sql queries, however one aspect I wasn't very familiar with was setting up a MySQL server and deploying it. Last time I tried, I found it very complicated and annoying(on windows at least) so I just ended up going with MongoDB since I didn't have to worry about hosting it
  • pytest - used for the unit tests. Even though I worked as a Software Engineer in Test at Ritual for roughly two years, I never wrote a single unit test till I got laid off. It was something I really wanted to know before I started a position elsewhere, so these days I'm writing unit tests for any project that I do. pytest came highly recommended on online forums and sites, and so far in my experience it's been pretty easy to grasp. I only have some basic tests written out right now, but am planning on adding some more tests later once the FE is done.
  • CircleCI - the CI for this project. I saw CircleCI mentioned on a lot of job postings so I decided to check it out. I love the debug with ssh feature. So far in this project, I've used it twice to quickly figure out what was going wrong with the pipeline. In the end, it took me about 30 tries in a day, to get the initial pipeline up and running but it was great feeling to see commit messages and PRs with a green checkmark next to it. Also having the
  • Bootstrap - the styling for this site.
  • unDraw - provided all of the gorgeous illustrations used in this site
  • Docker
